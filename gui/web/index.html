Online · THUG Pro

Play online via the THUG Pro total-conversion mod. This downloads THUG Pro's official installer and opens it; when it asks, point it at a clean THUG2 folder. Then choose "Online (THUG Pro)" above and press Play.

THUG Pro is a separate community app; its installer downloads the full build itself.

⟳ · Update

Update to the latest release + rebuild.

Installing replaces the toolkit binaries, then rebuilds. Your game data and saves are untouched. Restart this window afterwards.

⌫ · Uninstall

Remove Revert from this machine. Your saves and created tags are backed up to a dated folder in your home directory first.

The default keeps your saves, the Go build tool, and shared system libraries.
